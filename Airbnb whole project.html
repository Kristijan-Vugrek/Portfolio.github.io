For source of data I used Kaggle where I found "New York City Airbnb Market" dataset. After reviewing dataset and cleaning it in Excel, my main goal was to find some answers from this dataset.

Excel

Excel dashboard shows answers on questions:
- What is average price for each type of housing?
- What price is most frequently used?
- How many housings is in each New York borough and what is their price range?

Tableau

Tableau map shows answers on questions:
- What is average price for each type of housing?
- What is average, minimum, maximum price and number of housings by each boroughs?
- How many housings is in each New York borough?

But because Tableau map is interactive there is filter for filtering housing types
